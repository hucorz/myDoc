    1_What are we going to learn?¶

    System¶

    system 表示一个有 input 和 output 的系统，也可以叫做 function，transformation，operator

    比如：语言识别，语音助手，电话

    Two properties¶

    1. persevering multiplication
    2. persevering addition

    举个反例：\(input:x,\ output:x^2\) 就不是一个 Linear System

    这 2 个性质对多输入的 system 同样适用

    Application¶

    • Circuits

    电路中 input 是电压源或者电流源，output 是负载上的电源和电流

    • 信号与系统

    Fourier Transorm 其实是 Linear System，只是从 2 给不同的角度观看同一个信号

    • PageRank

    搜索引擎中对不同网页的排序，后面会细讲

    • Computer Graphics

    图形的变换，比如旋转

    2_System of linear equations¶

    System of Linear Equations：多元一次联立方程式

    术语¶

    定义域 && 值域¶

    Domain(定义域)：input 的集合

    Range(值域) 是 output 的集合，比 Co-domain(对应域) 小或者相同

    Co-domain 指的是对一个 function 不假思索的 output的范围，比如 \(y=x^2\)，input 是实数，不分析这个 function 的内涵， output 也一定是实数，所以 Co-domain 是实数

    一对一 && 映成¶

    One-to-one：一对一，单射

    Onto：Range 等于 Co-domain，满射

    线性系统 与 线性方程组¶

    只需要满足 Linear System 的两个 properties 就是线性系统

    Linear system 并不是一条斜着的直线，比如 微分 与 积分 同样也是线性系统（所以说 Fourier Transorm 是线性系统），因为他们满足两个 properties

    这门课研究的 Linear System 输入是一个 n 维的 向量，输出是一个 m 维向量，这种表示方式可以 apply 在非常多的地方

    这种 Linear System 可以写成 System of linear equations

    3_Vector¶

    如果不刻意说明，向量都认为是 列向量

    \(R^n\) 代表所有 n 维向量的集合，比如 \(R^2\) 表示整个 2 维平面

    4_Matrix¶

    讲 Vector 放在一起，就变成了 Matrix

    \[ A=[a_1,a_2,a_3]=\begin{bmatrix} 2 &3 &5 \\ 3 & 1 & -1\\ -2& 1& 1 \end{bmatrix} \]

    \(M_{n\times m}\) 表示所有 n 行 m 列的 Matrix（先 row 再 column）

    特殊矩阵：

    \(O\)：所有 element 都是 0 的矩阵，可以是任何的 size，\(O_{2\times3}\) 指定大小

    \(I\)：Identity matrix，单位矩阵，必须是正方形且左上到右下的元素是1，其他是 0，\(I_2\) 表示是 \(2\times 2\) 的大小

    5_Matrix-vector product¶

    假设 \(A\) 是 Matrix，\(x\) 是 Vector

    \(Ax\) 可以看成是多元联立方程式

    • 从行的角度：\(Ax\) 可以看成 Matrix 的每一行和 \(x\) 做内积组成的 Vector
    • 从列的角度：\(A\) 中的每列（每列是一个Vector）做 \(x\) 的线性组合

    如果 \(A\) 和 \(B\) 都是 \(m\times n\) 的 Matrix，\(Aw=Bw\) 对任意的 \(w\) 成立，\(A=B\)

    6_Having solution or not¶

    System of Linear Equations 是否有解？

    方程有解就说是 consistent（相容的），不然就是 inconsistent

    二维中 2 个非平行的 Vector 可以表示任何的 Vector（高维不一定），二维中非平行一定有解，有解不一定非平行

    Span¶

    一个 Vector Set \(S=\{v_1, v_2, ...\}\) 的所有 Linear Combination 的集合，叫做 Span S

    假如一个 Vector Set \(V\) = Span \(S\)，那 \(S\) 就是 \(V\) 的 generating set

    Standard Vector：标准向量

    有没有解的问题可以变成 \(A\) 的 所有 column 进行 Span 得到的 Vector Set V 中是否有 B

    7_How many solutions¶

    有解的情况下有多少解？

    唯一解：\(A\) 的 column 是 independent 或者 \(Rank\ A = n\) 或者 \(Nullity\ A=0\)

    无穷多解：\(A\) 的 column 是 dependent 等于 \(Rank\ A < n\) 或者 \(Nullity\ A>0\)

    其中 \(Rank+Nullity=n\)

    homogeneous：齐次的，若有一个 system linear equation 的 constant term 等于 0（\(b=0\)），那么就说是齐次的， homogeneous linear equation一定有解，至少有一组解是 Zero Vector

    geneous：非齐次

    证明有解时 dependent 有无穷多解：

    易证 \(Au=0\) 有无穷解，又 \(Av=b\) 有解，相加即可

    Dependent¶

    线性相关（dependent）与 线性无关（independent）

    一组 Vector Set 是 dependent 的满足：能找到一组 scalar 不全为 0，对 Vector Set 作 Linear Combination 后为 zero Vector，否则就是 independent

    Rank && Nullity¶

    Rank：Matrix 中最多的 independent columns 的数目

    Nullity：\(n-Rank\)

    8_Solving system of linear equations¶

    三种等价变化：

    1. Interchange any two rows of the matrix
    2. Multiply every entry of some row by the same nonzero scalar
    3. Add a multiple of one row of the matrix to another row

    RREF¶

    RREF：Reduced Row Echelon Form，简化列梯形矩阵

    Row Echelon Form：

    1. Each nonzero row lies above every zero row
    2. The leading entries are in echelon form

    RREF：

    1. Row Echelon Form
    2. The columns containing the leading entries are standard vectors

    Gaussian elimination：an algorithm for finding the reduced row echelon form of a matrix

    9_What can we know from RREF¶

    Augmented Matrix：增广矩阵

    RREF v.s. Linear Combination¶

    Column Correspondence Theorem：初等变换不改变列与列之间的线性组合关系

    假设 \(R\) 是 \(A\) 的 RREF，Span \(A\) = Span \(R\)

    RREF v.s. Independent¶

    pivot columns：RREF 中的 Leading entries 所在列对应到 \(A\) 的列，是 independent 的

    The non-pivot columns are the linear combination of the previous pivot columns

    Independent：Every column in RREF(A) is standard vector 或者 Every column is a pivot column

    矮胖型的 Matrix：行数 < 列数 => dependent

    More than m vectors（列向量） in \(R^m\) must be dependent

    以 \(2\times3\) 的矮胖型 Matrix 举例，可以理解为 input 3维，output 2 维，也就是降维，那么 output（b）在高维（input）上的映射就可以是无穷多个点

    RREF v.s. Rank¶

    Rank = Maximum number of Independent Columns

    ​ = Number of Pivot Column

    ​ = Number of Non-zero rows in RREF

    Properties of Rank from RREF：

    1. Rank A ≤ min(m, n)
    2. If m < n, the columns of A is dependent
    3. In \(R^m\), you cannot find more than m vectors that are independent

    Rank = Maximum number of Independent Columns 或者 Number of Pivot Column 或者 Number of Non-zero rows of RREF 或者 Number of Basic Variables

    Nullity = no. column - rank 或者 Number of Free Equations

    Nullity \(\not =\) Number of zero rows of RREF

    RREF v.s. Span¶

    Given \(Ax=b\)

    Rank A \(\not=\) rank [A b]

    Ax = b is consistent for every b

    \(\Harr\) RREF of [A b] cannot have a row whose only non-zero entry is at the last column

    \(\Harr\) RREF of A cannot have zero row

    \(\Harr\) Rank A = no. of rows

    10_Matrix Multiplication¶

    Definition¶

    \(C=AB\)，\(C\) 这个变换可以理解成 input 先经过 \(B\) 变换再经过 \(A\) 变化

    4 个不同的角度看 Matrix Multiplication

    • 行列，inner product，\(c_{ij}\) 为 \(A\) 的 \(i\) 行 与 \(B\) 的 \(j\) 列的内积

    • 列列，Linear combination of columns，对 \(A\) 中 column 做 \(B\) 的列数次不同的线性组合，然后横向组合起来

    • 行行，Linear combination of rows，对 \(B\) 中 row 做 \(A\) 的列数次不同的线性组合，然后纵向组合起来

    • 列行，summation of matrices，\(C=AB=a_1b_1^T+a_2b_2^T+...+a_nb_n^T\)，可以看成是一堆 Rank=1 的 Matrix 相加

    Block Multiplication

    分 Block 的方式：A 中列的划分和 B 中行的划分对应，A 中行的划分对应 B 中列的划分

    Property¶

    • \(AB\not=BA\)
    • \((AC)^T=C^TA^T\)
    • \(A(CP)=(AC)P\)，但是不同的方式运算量区别会很大

    Special Matrix¶

    Diagonal Matrix：对角矩阵

    Symmetric Matrix：对称矩阵

    11_Matrix Inverse¶

    Definition¶

    给定 2 个 function \(A\)，\(B\)，如果 for any \(v\)，\(g(f(v))=v\) 且 \(f(g(v))=v\)，那么 \(g\) 和 \(v\) 是互逆的

    对矩阵来说，如果 \(AB=I\) 且 \(BA=I\)，那 \(A\) 和 \(B\) 是互逆的

    利用逆矩阵求解 system of linear equations：\(Ax=b\Rightarrow x=A^{-1}b\)（手算的效率并不高，不如直接求 RREF）

    Property¶

    \(A\) 的互逆矩阵是唯一的，证明：有 \(AB=I,BA=I,AC=I,CA=I\)，那么 \(B=BI=B(AC)=(BA)C=IC=C\)

    如果 \(A\) 不是 square，那必不是 invertible 的，因为 \(AB\) 或者 \(BA\) 表示的 function 必定有一种情况是表示先从高维变化到低维然后变回高维，无法回到原样

    \((AB)^{-1}=B^{-1}A^{-1}\)

    \((A^T)^{-1}=(A^{-1})^T\)，可以证明 \(A\) 是 invertible 时，\(A^T\) 也是 invertible

    Invertible¶

    One-to-one 的情况下，那至多一个解（\(b\) 可能不在 Range 里）

    Onto 的情况下，永远有解（Co-domain 和 Range 一样大，\(b\) 永远在 Range 中）

    Domain 和 Co-domain 必须要同维度才是 invertible 的

    如果 \(A\) 是 Square 的，One-to-one \(\Leftrightarrow\) Onto

    \(A\) 是 invertible 等效于

    一般情况下检查是否 Invertible 都直接算 RREF 是否为 \(I\)

    如果 \(A_{n\times n}\) 存在 \(BA=I\)，那 \(A\) 是 invertible 的，\(AB=I\) 一样

    利用反证法，假设 存在 \(v\not= 0\) 使 \(Av=0\)

    Inverse of Elementary Matrices¶

    Elementary Matrices：初等矩阵，即初等行变换对应的矩阵

    这些 Matrix 的 inverse 按照其初等行变换的意义即可算出其 inverse

    invertible 的 Matrix 相乘仍然是 invertible 的

    invertible 的 Matrix 可以写成 product of elementary matrix

    Inverse of General Matrices¶

    通过 \(E_1...E_nA=R\) 得到 \(A\) 的 inverse，其中 \(R\) 是 RREF

    方法是 Transform \([A\ I_n]\) to its \([R\ B]\)，其中 \(B=E_n...E_1\)，就是 \(A\) 的 inverse

    如果要求 \(A^{-1}C\)，那把上面的 \([A\ I_n]\) 变成 \([A\ C]\) 即可

    14_Column Space, Null Space, Row Space¶

    三个 Space 所在的维度：

    三个 Space 的 Basis：

    Col A：

    The pivot columns of A form a basis for Col A

    Basis：\(Col \ A=Span\{pivot\ columns\}\)

    Dim(Col A) = number of pivot columns = rank A

    Null A:

    Basis:

    • Solving Ax = 0

    • Each free variable in the parametric representation of the general solution is multiplied by a vector.

    • The vectors form the basis.

    Dim (Null A) = number of free variables = Nullity A = n - Rank A

    Row A：

    Basis: Nonzero rows of RREF(A)

    Dim (Row A) = Number of Nonzero rows = Rank A

    Rank 能代表的含义：

    \[ Rank\ A = Dim(Col\ A) \\ Rank\ A = Dim(Row\ A) = Dim(Col\ A^T)=Rank\ A^T \]

    可以得到 \(Rank\ A = Rank\ A^T\)

    Distance：\(||u-v||\)

    Dot Product：点乘

    Orthogonal：if \(u\cdot v=0\)，\(u\) and \(v\) are Orthogonal，and \(||u+v||^2=||u||^2+||v||^2\)（勾股定理）

    ​ \(||u+v||^2=||u||^2+||v||^2+2(u\cdot v)\)（余弦定理）
