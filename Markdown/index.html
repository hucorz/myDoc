    Markdown

    一种方便简单的排版语言

    不同的平台使用 Markdow 规则可能有些许不同，比如标题 # 后面可能不需要加空格，本文主要基于 typora

    标题

    一级标题 # + 空格

    二级标题 ##+ 空格

    ...最多有六级标题

    强调

    *和_都可以表示强调

    使用方式：在需要强调的部分的头和尾都要加

    • 一个*或者_表示倾斜
    • 两个*或者_表示加粗
    • 三个*或者_表示加粗＋倾斜

    超链接与图片

    行内式

    格式：[名字] （URL）

    [百度](https://www.baidu.com/)

    效果如下：

    百度

    参考式

    在要多次使用同一个超链接时使用

    格式：[给网站起的名字]：URL （相当于定义）

    使用方式：[超链接的名字] [给网站起的名字]

    [百度]:https://www.baidu.com/
[here][百度]

    图片

    也有行内式与参考式，只是在前面加个! 

    ![图片名][具体地址，本地或者网页]

    \(\lim_{n\rightarrow+\infty}\) \lim_{n\rightarrow+\infty}

    \(\sum_{i=1}^{n}{a_i}\) \sum_{i=1}^{n}{a_i}

    折叠代码和侧边栏

    typora中折叠侧边栏：文件-偏好设置-外观-侧边栏

    折叠代码：

    <details>
<summary>tag</summary>
<code>
Hello world!
</code>
</details>

    效果如下：

    tag Hello world!
