    ndarray¶

    创建 ndarray¶

    import numpy as np
np.array(object, dtype = None, copy = True)
np.empty(shape, dtype=float, order='C', *, like=None) # 并不初始化为0
np.ones(shape, dtype=None, order='C', *, like=None) # 初始化为 1
np.zeros(shape, dtype=float, order='C', *, like=None) # 初始化为 0
a = np.arange(24).reshape((2,3,4)) # range创建list,arange创建array
a = np.arange(24).reshape((2,3,-1)) # 参数有-1时会自动根据其他的参数计算
    名称 描述
    object 数组或嵌套的数列
    dtype 数组元素的数据类型，可选
    copy 对象是否需要复制，可选
    order 创建数组的样式，C为行方向，F为列方向，A为任意方向（默认）
    subok 默认返回一个与基类类型一致的数组
    ndmin 指定生成数组的最小维度

    数据统计：均值/求和/标准差/方差/协方差¶

    # 求平均值 axis=0 求列均值, axis=1 求行均值，默认求所有数的均值
np.mean(matrix, axis=None)

# 求和
np.sum(matrix, axis=None)

# 求标准差
np.std(matrix, axis=None)

# 求方差
np.var(matrix, axis=None)

# 求协方差 rowvar=True以行为变量，否则以列为变量
np.cov(matrix, rowvar=True)

    矩阵转置¶
    # 转置 不改变原矩阵
a.T() / np.transpose(a)

    a.T 的 a 一定要是 ndarray, 如果 a 是 list, 只能用 np.transpose(a) 且返回的是 ndarray,

    如果 a 是一维的 ndarray 不起作用

    如果想让一个 1n 维的 ndarray 或 list 变成 n1 维, 有 2 个方法

    法一：

    a = np.transpose([a]) # [a] 是一个 list

    法二：

    a = np.array([a])
a = a.T

    matmul()

    np使用之np.matmul_alwaysyxl的博客-CSDN博客

    np.matmul(a, b, out=None)
# 如果都是二维的，就是普通的矩阵乘法

# 如果某个参数是 N(n>2) 维的，该参数呗理解为一些矩阵(维数为最后2维)的stack,计算时会得到相应的广播
# eg：a:(2,2,4)和b:(2,4,2), a会被理解为2个(2,4),b会被理解为2个(4,2),结果是a的第一个和b的第一个，a的第二个和b的第二个相乘，结果为(2,2,2)
# a:(2,2,4)和c:(1,4,2)，c会被广播成2个，a中的2个分别与c相乘，结果为(2,2,2)

    数学运算¶

    np.power(a, 2) # 对a中每个元素平方, 而不是计算 a*a, 可以简写为 a**2
np.power(a, b) # a 与 b 的列数要相同, 对 b 中的每个元素, 计算 a[j] 的 b[i][j]次方

# 除了次方外，其他的运算同理，比如如果要对每个数字进行 log2 的操作不能使用 math.log2(),类似的有
np.log2(a)
np.around(a)
np.abs(a)
