    griddata¶

    from scipy.interpolate import griddata
grid_x, grid_y = np.mgrid[0:1:100j, 0:1:100j]
grid = griddata(points, values, (grid_x, grid_y), method='linear', fill_value=nan)

    如果是在二维上作图（ (X,Y) -> Z ），point 的维数是 [data_size, 2]，values 是 [1, data_size]

    method：表示插值方式，有 nearest / linear / cubic

    fill_value：设置外值，默认 non，对 nearest 不起作用
