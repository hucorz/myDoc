    Conda常用命令¶

    Conda常用命令及如何换镜像源 - 知乎

    conda activate xxxx    #开启xxxx环境
conda deactivate       #关闭环境

# 下面3个的作用相同
conda env list         #显示所有的虚拟环境
conda info --envs      #显示所有的虚拟环境
conda info -e          #显示所有已经创建的环境

conda update -n base conda            #update最新版本的conda

conda create -n xxxx python=3.6       #创建python3.6的叫做xxxx虚拟环境

conda remove --name xxxx  --all       #彻底删除旧环境
conda remove -n xxxx --all            #彻底删除旧环境

#Conda是没有重命名环境的功能, 要实现这个基本需求, 可以通过克隆-删除的过程。
#切记不要直接mv移动环境的文件夹来重命名, 会导致一系列无法想象的错误的发生!
conda create --name newname --clone oldname //克隆环境
conda remove --name oldname --all           //彻底删除旧环境

conda list                 #查看已经安装的文件包
conda list -n xxx          #指定查看xxx虚拟环境下安装的package
conda update xxx           #更新xxx文件包
conda uninstall xxx        #卸载xxx文件包

    argparse¶

    parse_args()和parse_known_args()的用法和区别

    解析命令行参数

    parser = argparse.ArgumentParser()
parser.add_argument()
opt = parser.parse_args()
opt = parser.parse_known_args()

    parser.add_argument()

    parser.add_argument()

    • name or flags - 一个命名或者一个选项字符串的列表，例如 foo 或 -f, --foo

    • action - 当参数在命令行中出现时使用的动作基本类型，默认 'store'，表示保存输入的值

    可选，"store_true" / "store_false"：如果配置值为 True 或 False，不能配置具体的值；

    • nargs - 命令行参数应当消耗的数目

    可选，N：非负整数，必须有N个参数；'?'：0个或1个参数；'*'：0个或多个；'+'：一个或多个；

    • default - 当参数未在命令行中出现并且也不存在于命名空间对象时所产生的值

    • type - 命令行参数应当被转换成的类型

    • required (bool) - 此命令行选项是否可省略 （仅选项可用）

    • help (str) - 一个此选项作用的简单描述

    parser.parse_args()

    返回一个命名空间，如果用了没有 add_argument() 的参数会报错

    parser.parse_args()
# 代码
import argparse
parser = argparse.ArgumentParser()
parser.add_argument('--flag_int', type=int, default=2, help='flag_int')
opt1 = parser.parse_args()
# 命令行
python test.py --flag_int 3
# 输出
Namespace(flag_int=3)

    parser.parse_known_args()

    返回一个元组，第一个元素是命名空间，其中有已经定义的参数，第二个元素是列表，以空格为间隔存储未定义的参数

    parser.parse_known_args()
# 代码
import argparse
parser = argparse.ArgumentParser()
parser.add_argument('--flag_int', type=int, default=2, help='flag_int')
opt1 = parser.parse_known_args()
# 命令行
python test.py --flag_int=1 --flag_float=1.0 --flag_bool=true
# 输出
(Namespace(flag_int=1), ['--flag_float=1.0', '--flag_bool=true'])
# 命令行
python test.py --flag_int 1 --flag_float 1.0 --flag_bool true
# 输出
(Namespace(flag_int=1), ['--flag_float', '1.0', '--flag_bool', 'true'])

    Scipy¶

    插值¶

    griddata¶

    from scipy.interpolate import griddata
grid_x, grid_y = np.mgrid[0:1:100j, 0:1:100j]
grid = griddata(points, values, (grid_x, grid_y), method='linear', fill_value=nan)

    如果是在二维上作图（ (X,Y) -> Z ），point 的维数是 [data_size, 2]，values 是 [1, data_size]

    method：表示插值方式，有 nearest / linear / cubic

    fill_value：设置外值，默认 non，对 nearest 不起作用
