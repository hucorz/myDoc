    CSS¶

    Cascading Style Sheets：层叠样式表

    和 HTML 一样也是一种标记语言

    作用：美化 HTML 文档，实现结构（HTML）与样式（CSS）相分离

    选择器分类：基础选择器 和 复合选择器

    基础概念¶

    CSS 规则的构成：选择器(HTML标签) + 一条或多条声明

    一般写在 \(<head>\) 中

    eg：

    <head>
    <style>  /*style 可以写在任何地方*/
        p {  
            color: red;  /*段落设置为红色*/
            font-size: 12px;  /*字体大小12像素*/
        }
    </style>
</head>

    CSS引入方式¶

    • 行内式，在标签的属性里直接添加样式 \(<p\ style="\ color:red;\ ">\)，适合修改简单样式，没有与结构分离

    • 嵌入式，写在 html 页面内部，即 \(<style></style>\) 中，并没有完全与结构分离

    • 链接式，单独写在 CSS 文件中，适用于样式比较多的时候，实现了与结构分离

    /*.css文件中*/
不需要写 <style></style>, 只需要写样式
/*.html文件中*/
<link rel="stylesheet" href="css文件路径">

    基础选择器¶

    基础选择器由单个选择器组成

    标签选择器¶

    标签名作为选择器

    标签名 {
    属性1: 属性值1;
    ...
}

    类选择器¶

    可以单独选择一个或几个标签，一个标签也可以有多个类名，用 . 来定义

    .类名 {
    属性1: 属性值1;
    ...
}

<div class="类名"> eg </div>
<div class="类名1 类名2"> </div>

    id选择器¶

    为标有特定 id 的 HTML元素 指定样式，用 # 来定义

    与类选择器的区别在于，id选择器只能调用一次，即 id 是独一无二的

    #类名 {
    属性1: 属性值1;
    ...
}

<div id="id"> eg </div>

    通配符选择器¶

    为页面中所有的元素指定样式，用 * 来定义

    不需要调用

    * {
    属性1: 属性值1;
    ...
}

    复合选择器¶

    后代选择器¶

    又称包含选择器

    # 元素 可以是任何基础选择器
元素1 元素2 {样式声明}
# 此样式只会应用于 元素1 中所有的 元素2（不一定正好是下一级），即后代元素

元素1 元素2 元素3 ... {样式声明}
# 后代可以有多层, 但只会应用于最后一层

# eg: 类选择器 + li + a
.nav li a {
    color: red;
}

    子选择器¶

    选择某元素最近一级的子元素

    元素1 > 元素2 {样式声明}
# 此样式只会应用于 元素1 中最近的 元素2, 元素2 必须是儿子

    并集选择器¶

    伪类选择器¶

    用于添加一些选择器的特殊效果，如为超链接的不同状态添加不同的颜色

    常用属性总结¶

    字体属性¶

    字体，大小，粗细，文本样式

    属性 含义 属性值举例
    font-family 字体 "Microsoft YaHei"
    font-size 大小 20px（不会影响标题）
    font-weight 粗细 700（加粗:100，普通:100）
    font-style 文字样式 normal（斜体:italic）

    复合属性

    /*font: font-style font-weight font-size/line-height font-family*/
/*上面的顺序不能更改*/
/*font-size 和 font-famliy 不可以省略*/
font: italic 700 16px 'Microsoft yahei';

    文本属性¶

    文本的外观属性

    属性 含义 属性值举例
    color 颜色 red / #FF0000
    text-align 水平对齐 center / left / right
    text-decoration 装饰文本 none / underline / overline / line-through
    text-indent 首行缩进 10px / 2em（2个当前文字大小）
    ling-height 行间距 26px

    装饰属性取 none 常用在取消超链接的下划线

    其他¶

    属性名 含义
    width / height 高度 / 宽度
    background-color 背景颜色
    margin
    padding

    Emmet 语法¶

    前身是 Zen coding，能够使用缩写提高 html / css 的编写速度，vscode 内部集成

    快速生成 HTML 语法

    • 生成标签：标签名 + tab
    • 生成多个标签：标签名*个数+tab，如 div*3+tab
    • 父子级关系标签，用 >，比如 ul>li+tab
    • 兄弟关系标签，用 +，比如 dip+p+tab
    • 带类标签，用 标签名.类名，比如 p.demo+tab，不写标签名默认标签是 div
    • 带 id 标签，用 标签名#类名，不写标签名默认是 div
    • 生成有序类名，.类名$*数字，会生成 类名1，类名2，，，
    • 标签内默认文字，用{}，如 div{eg}+tab，div{$}*5会生成5个自带12345的div

    快速生成 CSS 语法

    缩写+tab，如：

    tac+tab 表示 text-align: center;

    w100+tab 表示 width: 100px;

    tx2em+tab 表示 text-indent: 2em;
