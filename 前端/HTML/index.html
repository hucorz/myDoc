    HTML¶

    基础概念¶

    HTML¶

    HTML 是一种超文本标记语言（即可以加入多媒体内容，超越了文本限制，还支持超链接）

    WEB标准¶

    构成：结构（HTML），表现（CSS），行为（JS）

    VSCODE使用¶

    插件¶

    插件 作用
    open in browser 右键即可在浏览器中打开
    Auto Rename Tag 自动补全双标签

    快速格式化¶

    快速格式化文档： ALT SHIFT F

    保存文件自动格式化：在 setting.json 中加上

    "editor.formatOnType": true, // HTML 文件保存自动格式化 
"editor.formatOnSave": true

    标签¶

    基本结构标签¶

    标签名 说明
    \(<html></html>\) 根标签
    \(<head></head>\) 头部标签
    \(<title></title>\) 标题标签
    \(<body></body>\) 主体标签
    \(<!DOCTYPE html>\) 声明html版本
    \(<html lang="zh-CN">\) 声明网页语言
    \(<meta charset="UTF-8">\) 声明字符集为万国码

    标题标签¶

    最多有6级标题标签

    <h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>

    段落和换行标签¶

    标签名 说明
    \(<p></p>\) 段落标签，paragraph
    \(<br>\) 换行标签，break

    文本格式化标签¶

    标签名 说明
    \(<strong></strong>\)或者\(<b></b>\) 加粗
    \(<em></em>\)或者\(<i></i>\) 倾斜
    \(<del></del>\)或者\(<s></s>\) 删除线
    \(<ins></ins>\)或者\(<u></u>\) 下划线

    div 和 span 标签¶

    这2个标签都是布局用

    标签名 说明
    \(<div></div>\) 分区，占一行，division
    \(<span></span>\) 跨距，可以多个占一行，

    图像标签¶

    <img src="路径或者链接"/>

    img 标签中可加的属性

    属性 说明
    src 路径或链接，source
    alt 显示失败时的替换文本，alternative
    title 鼠标放在图片上时的说明文字
    width 设定宽度
    height 设定高度
    border 设定边框粗细

    宽度和高度只设定一个时纵横比是固定的

    超链接标签¶

    <!-- anchor -->
<!-- target="_self"时不开新窗口  "_blank"时开新窗口 -->
<a herf="跳转目标" target="窗口弹出方式"> 文本或图像标签 </a>  
<!-- herf 是html文件时是内部链接，是网页链接时是外部链接 -->
<!-- herf = "#" 表示 空链接-->
<!-- herf = "文件路径" 时可以实现下载文件-->

    锚点链接

    <!-- 出发位置 -->
<a herf="#tag"> 文本或图像标签 </a>
<!-- 目标位置在标题处 -->
<h2 id ="tag">标题</h2>

    注释和特殊字符¶

    <!-- 这是注释标签，vscode 用 Ctrl+/ -->

    特殊字符：

    HTML 符号实体参考手册 | 菜鸟教程

    常用：

    | 描述      | 代码        |
| -------- | ----------- |
| 空格      | &nbsp;     |
| 大于      | &gt;       |
| 小于      | &lt;       |
| 与号      | &amp;      |
| 乘法      | &times;    |
| 除法      | &divide;   |

| 立方      | &sup3;     |

    表格标签¶

    <table>  <!-- 表格根标签 -->
    <tr>   <!-- 表格中的行 -->
        <td>单元格内的文字</td> <!-- 每行中每格的内容 -->
        ...
    </tr>
    ...
</table>

<!-- 表头单元格标签，指明某一行是表头，和td同级，会加粗并居中 -->
<th>...</th>

    表格属性

    不太常用，会通过CSS设置

    属性名 属性值 描述
    align left / center / right 相对周围元素的对齐方式
    border 1 / "" 是否有边框，""为默认，表示没有
    cellpadding 像素值 边沿与内容之间的空白，默认为1
    cellspacing 像素值 单元格之间的空白，默认2
    width 像素值 / 百分比 宽度
    height 像素值 / 百分比 高度

    表格结构标签

    <table>
    <thead></thead>   <!-- 头部标签 -->
    <tbody></tbody>   <!-- 主体标签 -->
</table>

    合并单元格（td的属性）

    • 跨行合并：rowspan="合并单元格的个数"，最上侧单元格为目标单元格
    • 跨列合并：colspan="合并单元格的个数"，最左侧单元格为目标单元格

    列表标签¶

    分类：无序列表，有序列表，自定义列表

    列表自带的样式属性可以通过CSS修改

    无序列表

    <ul>   <!-- ul中只能放li标签，但li中可以放任何标签 -->
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
    ...
</ul>

    有序列表

    <ol>   <!-- ol中只能放li标签，但li中可以放任何标签 -->
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
    ...
</ol>

    自定义列表

    <dl>  <!-- dl中只能放dt/dd标签，但dt/dd中可以放任何标签 -->
    <dt>名词1</dt>
    <dd>名词1解释1</dd>
    <dd>名词1解释2</dd>
</dl>

    表单标签¶

    表单用于收集用户信息

    表单由 表单域，表单控件(表单元素)，提示信息 构成

    <form>
</form>
    属性 属性值 作用
    action url地址 指定接受并处理表达那数据的服务器程序的url地址
    method get/post 设置表单数据的提交方式
    name 名称 设置表单名称

    input 输入表单元素¶

    <input type="" /> <!-- type指定不同的控件类型 -->
<!-- eg -->
<!-- 单选属性必须有相同的name值 -->
性别：男 <input type="radio" name="sex"> 女 <input type="radio" name="sex">
    type属性值 描述
    button 点击按钮（多数情况下通过JS启动脚本）
    checkbox 复选框
    file 输入字段和"浏览"按钮，供文件上传
    hidden 隐藏的输入字段
    image 图像形式的提交按钮
    password 密码字段，暗文输入
    radio 单选按钮
    reset 重置按钮
    submit 提交按钮
    text 输入字段，宽度默认为20字符
    input其他属性 属性值 描述
    name 自定义 名称
    value 自定义 input的元素值
    checked checked 规定此input元素首次加载时应当被选中
    maxlength 正整数 输入字段字符的最大长度

    label 标签¶

    绑定一个表单元素，当用户点击 label 标签内的文本时，会自动对应到对应的表单元素上

    <label for="sex">男</label>
<input type="radio" name="sex" id="sex" />
<!-- for 与 id 的属性值要相同 -->

    select 下拉表单元素¶

    select 中至少有一个 option，option

    <select>
    <option>选项1</option>
    <option>选项2</option>
    <!-- selected 表示默认选中项 -->
    <option selected="selected">选项3</option>
    ...
</select>

    textarea 文本域元素¶

    用于输入多行的文本输入的控件

    <!-- 实际开发中都是用CSS来改变大小 -->
<textraea rows="3" cols="20">
    文本内容
</textraea>

    其他标签¶

    button¶

    <button>搜索</button>
/*等效于<input type="button" value="搜索">*/
