    Keras¶

    Why Keras¶

    tensorflow 和另外一个功能相近的 toolkit theano 很难在短时间内精通

    Keras 是比较容易去学习和使用的，并且它也有足够的弹性，除非你自己想要做 deep learning 的研究，去设计一个自己的 network，否则多数你可以想到的 network，在 Keras 里都有现成的 function 可以拿来使用；因为它背后就是 tensorflow or theano，所以如果你想要精进自己的能力的话，你永远可以去改 Keras 背后的tensorflow 的 code，然后做更厉害的事情

    而且，现在 Keras 已经成为了 Tensorflow 官方的 API，它像搭积木一样简单

    prepare data¶

    使用的data是MNIST的Data：http://yann.lecun.com/exdb/mnist/

    Keras提供了自动下载MNIST data的function：http://keras.io/datasets/

    MNIST 是手写数字的数据集

    process¶

    DOC：Keras API reference

    首先要先导入keras包：from keras.models import Sequential

    step 1：define a set of function——neural network¶

    先用Sequential()宣告建立一个model

    model = Sequential()

    然后开始叠 layer 以构成一个 neural network，这里的例子一共有2层，每层 500 个 nuits（units）

    model.add(Dense(input_dim=28 * 28, units=500, activation='sigmoid'))
model.add(Dense(units=500, activation='sigmoid'))
model.add(Dense(units=10, activation='softmax'))

    add：加一层 hidden layer

    Dense：加一个 Fully connected 的 layer

    input_dim：只在第一层 layer 前面加一个 input，这里是一个 28*28=784 的 vector，代表 img

    units：一层 layer 中 units(neuron) 的数量

    activation：激活函数，一些可选的例子：softplus, softsign, relu, tanh, hard_sigmoid, linear

    最后一层 output 是 10 维，分别表示数字 1~10，用 softmax 使得几率之和变成一个概率分布

    Step 2：goodness of function——cross entropy¶

    利用 model.compile 去定义 loss function

    model.compile(loss='categorical_crossentropy',
              optimizer='adam',
              metrics=['accuracy'])

    loss: 使用的 loss function

    optimizer: 系数的更新方式，一些可选的例子：Adagrad, Nadam

    Step 3：pick the best function¶

    用 model.fit 去 training

    model.fit(x_train, y_train, batcha_size=100, nb_epoch=20)

    x_train：前 2 个参数是 train_data，以手写数字辨识为例，一张图片是 28*28=784 的 vector，样本量为 10000，那么 x_train 就一个 10000*784 的 numpy array；y_train 是一个 10000*10 的 numpy array

    batch_size：batch 的大小，即每个随机选择多少个样本来 update 参数

    nb_epoch：对每个 batch update 参数的次数，以上面的参数为例，10000 个 data，100 个为一个 batch，则有 100 个 batch，所以一共 update 2000 次（一个 epoch 中每个 batch 只 update 一次参数）

    如果 batch_size=0 就是 Stochastic Gradient Descent (随机梯度下降法)

    运用了 GPU 的平行运算后，batch_size=10 和 batch_size=1 在 update 一次参数上的速度基本是一样的，但 batch_size 也不能开太大，GPU 是有极限的。而且在 gradient descent 的时候可能会卡在 local minima

    Save and Load Models¶

    Keras 可以把 train 好的 model 保存起来并直接使用保存好的 model，参考：Keras FAQ

    Testing and Predict¶

    用 model.evaluate 计算正确率

    score = model.evaluate(x_test,y_test)
print('Total loss on Testing Set:',score[0])
print('Accuracy of Testing Set:',score[1])

    用 model.predict 来做 predict

    result = model.predict(x_test)

    Appendix：手写数字识别完整代码¶

    import numpy as np
from keras.models import Sequential
from keras.layers.core import Dense, Dropout, Activation
from keras.layers import Conv2D, MaxPooling2D, Flatten
from keras.optimizers import SGD, Adam
from keras.utils import np_utils
from keras.datasets import mnist

# categorical_crossentropy
def load_data():  
    (x_train, y_train), (x_test, y_test) = mnist.load_data()
    number = 10000
    x_train = x_train[0:number]
    y_train = y_train[0:number]
    x_train = x_train.reshape(number, 28 * 28)
    x_test = x_test.reshape(x_test.shape[0], 28 * 28)
    x_train = x_train.astype('float32')
    x_test = x_test.astype('float32')
    # convert class vectors to binary class matrices
    y_train = np_utils.to_categorical(y_train, 10)
    y_test = np_utils.to_categorical(y_test, 10)
    x_train = x_train
    x_test = x_test
    # x_test=np.random.normal(x_test)
    x_train = x_train / 255
    x_test = x_test / 255

    return (x_train, y_train), (x_test, y_test)


if __name__ == '__main__':
    # load training data and testing data
    (x_train, y_train), (x_test, y_test) = load_data()

    # define network structure
    model = Sequential()

    model.add(Dense(input_dim=28*28, units=500, activation='sigmoid'))
    model.add(Dense(units=500, activation='sigmoid'))
    model.add(Dense(units=10, activation='softmax'))

    # set configurations
    model.compile(loss='categorical_crossentropy',
                  optimizer='adam', metrics=['accuracy'])

    # train model
    model.fit(x_train, y_train, batch_size=100, epochs=20)

    # evaluate the model and output the accuracy
    result = model.evaluate(x_test, y_test)
    print('Test Acc:', result[1])

    Test Acc: 0.9439
