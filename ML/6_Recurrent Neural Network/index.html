    Recurrent Neural Network¶

    RNN：循环神经网络

    Basic Concept¶

    Slot Filling：Slot Filling 希望实现的是给出一个句子，如 I would like to arrive Taipei on November 2nd 后，能够得到其中 2 个主要内容，Destination：Taipei，time of arrival：November 2nd，所以叫 Slot Filling（槽填充）

    input 因为是单词所以可以用 1-of-N encoding，除此之外，我们还要加一个 dimension for "Others"，也可以用 Word hashing

    1111

    但还有一个问题就是，并不一定都是 arrive 的，也有可能是 leave Taipei 的，所以我们要给 Neural network 更多的 memory，去记住前面的信息

    可以用 2 个额外的 memory 去 store 前面的信息，并把他们也作为input

    以下图为例，memory 的初始值设为 0，Neural network 的 weight 都是 1，bias 都是 0，第一层 layer 的 output 会存储到 memory 中作为下一次的 input

    22222

    大体的流程如下（同一个 network 使用多次）

    3333

    当然也可以是 deep 的，每一层的 layer 的 output 都存储下来

    4444

    上面的叫做 Elman Network，还有其他不同 sturcuture 的 network 比如 Jordan Network 能得到更好的结果，因为它的 output \(y^t\) 是有 target 的​

    5555

    Bidirectional RNN

    RNN 也可以是双向的，正向做的同时逆向做，并得到 \(y^t\)​​​，这样的话 Neuron Network 能够同时看两边

    6666

    Long Short-term Memory（LSTM）

    每个 Memory Cell 都有一个 input gate，output gate，forget gate，什么时候打开 input gate，打开 output gate，打开 forget gate 都是 network 自己学到的

    可以看成有 4 个 input，1 个 output

    77777

    \(c^{'}\)​ 是新的存储在 memory 中的值

    8888

    下图是个例子，\(x_1\) 表示 input data,\(x_2 = 1\) 表示 input gate 打开，\(x_2=-1\) 表示 forget gate 打开，\(x_3=1\)​​ 表示 output gate 打开，每一个 weight 和 bias 都是 train 出来的

    LSTM 相比于普通的 Network 需要 4 倍的参数

    999

    LSTM 的真正形态：除了当前时间点的 input 外还有上一个时间点的 output 和 peephole（把存在 memory 中的值也加进 input）

    Keras 中支持三种 LSTM：LSTM，GRU，SimpleRNN

    How to learn?¶

    Loss Function¶

    每一个 output 的与 label 的 cross entropy 的和

    1111

    update 的方法也是用 backpropagation，叫做 backpropagation through time（BPTT），在 time sequence 上运作，这里不展开

    不幸的是，RNN-based network 是不太容易 train 的，total loss 并不是一直下降的而且在一些地方非常的平坦，一些地方非常的陡峭

    1212

    Clipping¶

    通过 Clipping 解决 total loss 的图像陡峭的问题

    Clipping 是在 update 的过程中当 gradient 大于某个值时，让它就等于某个值，即设定一个 threshold

    会出现这样的原因不是因为 activation function，当把 sigmoid function 改成 ReLU 后 performance 反而会变差

    下面用一个最简单的例子来解释为什么会有这种情况

    当除了 neuron transition 之间的 weight 都是 1 时，第一个 input 改变一点，对 ouput 的影响就会非常大，learning rate 太大太小都会有问题

    1313

    解决这个问题被广泛采取的方法就是 LSTM，它可以把 loss 图像中平坦的地方拿掉（避免 gradient 特别小），这样就可以放心的把 learning rate 设小一些；因为 如果 weight 的值对 memory 的产生了影响，这个影响会一直存在（除非 forget 掉）

    Gated Recurrent Unit（GRU）simpler than LSTM 只有 2 个 gate，所以参数比 LSTM 少，如果 overfitting 了可以尝试
