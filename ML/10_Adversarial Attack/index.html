    Attack ML Models¶

    Motivation¶

    应对来自人类的恶意

    对一张图片加入一些杂讯会对 Classifier 的结果造成影响

    Attack¶

    Attack 就是让一张图片不改变人类对它的辨识的情况下，改变同一个 model 的结果

    Loss Function for Attack¶

    在 train 一个 model 时，input \(x\)​ 是固定的，loss function为 \(L_{train}(\theta)=C(y^0,y^{true})\)​，通过不断调整 \(\theta\)​ 的值，使得 \(L_{train}(\theta)\)​​​ 越小越好

    Non-targeted Attack：此时 network 的参数 \(\theta\)​ 是固定的，通过不断调整 input \(x\)​ 的值使得 \(L(x')=-C(y',y^{true})\)​​​ 越小越好​，即和原来的 target 越远越好

    Targeted Attack：此时 network 的参数 \(\theta\) 是固定的，通过不断调整 input \(x\) 的值使得 \(L(x')=-C(y',y^{true})+C(y',y^{false})\)​​​ 越小越好，即和原来的 target 越远越好，和新的 target 越近越好

    与此同时还需要限制输入 \(x^0,x'\)​ 之间的差异（不改变人类对它的辨识的情况），只有这输入之间的差异 d 小于 \(\epsilon\)​，我们才可以认为 \(x'\)​ 是与 \(x^0\)​ 相似的，才达到了 attack 的目的，即使输入尽可能具有迷惑性，从而使网络输出错误的结果

    Constraint¶

    计算 d 的方法

    • L2-norm，为 \(x^0,x'\) ​之间每个像素差异的平方和
    • L-infinity，为 \(x^0,x'\)​​ 之间每个像素差异的最大值

    如果改变一张图中的每个 pixel，另外一幅图只改变其中一个 pixel，使得这两者之间的L2-norm 是一样的，但第二种方式得出的 L-infinity 更大

    How to Attack¶

    就像训练一个 neural network 一样，但需要训练的参数是 \(x'\)，此时就需要找到一个参数 \(x^*\)，来最小化 \(L(x')\)，限制条件是 \(d(x^0,x')\leq\epsilon\)​ $$ x^*=arg\mathop{min}_\limits {d(x^0,x')\leq\epsilon}L(x') $$

    当 \(d(x^0,x^t)>\epsilon\) 时，，使用 \(fix(x^t)\) 来更新，如下图所示，如果超过了 \(\epsilon\) 那就设为 \(\epsilon\)

    假设 \(x_0\) 是在高维上移动的一个点，在某些维度上移动时，它还是原来的类别的可能性还是很高，但在某贴特定的维度上移动时，只需要移动一点点就会变成一个完全不一样的类别

    Attack Approaches¶

    • FGSM (https://arxiv.org/abs/1412.6572)
    • Basic iterative method (https://arxiv.org/abs/1607.02533)
    • L-BFGS (https://arxiv.org/abs/1312.6199)
    • Deepfool (https://arxiv.org/abs/1511.04599)
    • JSMA (https://arxiv.org/abs/1511.07528)
    • C&W (https://arxiv.org/abs/1608.04644)
    • Elastic net attack (https://arxiv.org/abs/1709.04114)
    • Spatially Transformed (https://arxiv.org/abs/1801.02612)
    • One Pixel Attack (https://arxiv.org/abs/1710.08864)
    • ...... only list a few

    这些方法的主要区别在于使用了不同的 constrains，或者使用了不同的 optimization methods

    这里介绍一下FGSM，它的策略是对每一维的 \(\frac{\partial L}{\partial x_i}\) 如果是正值取 +1，负值就取 -1；即对于 \(x^0\) 的所有维，要么\(+\epsilon\)，要么\(-\epsilon\)​；这样的话在FGSM里面，我们只在意 gradient 的方向，不在意其具体的大小

    Black Box Attack¶

    攻击一个黑箱，即不知道具体参数的 neuron network，如果知道 black network 的 training data，我们可以利用 data 来 train 一个 proxy network，再利用这个 proxy network 来 attack

    如果得不到 training data 但是 network 是在线版本，可以输入大量图片来制造 data

    这样的方法是很有可能奏效的

    Defense¶

    Adversarial Attack cannot be defended by weight regularization, dropout and model ensemble

    Two types of defense:

    • Passive
    • Proactive

    Passive Defense¶

    图片丢进 network 前丢进一个 filter，这个 filter 可以是 smoothing

    Q：那么为什么 smoothing 可以达到这种效果呢？

    A：只有某几种方向上的的信号可以使 attack 成功。如果使用了 smoothing 这种 filter，就把这几种信号改变了，那么 attack 就失效了 ；加上 smoothing 并不会伤害原来的图片，所以 network 仍然可以得出正确的结果

    根据这种思想，有学者就提出了 feature squeeze，Squeezer 就是不同的 filter

    对于同一个 input，我们先得出 model 的输出结果 \(Prediction_0\)​，再根据 \(Squeeze_1，Squeeze_2\)​ 得出结果 \(Prediction_1,Prediction_2\)​，如果 \(Prediction_0\) ​和\(Prediction_1,Prediction_2\)​​​ 之间的差值 d 很大，那么我们就可以认为 input 是来 attack 的

    另外一个方法

    对原图片进行缩放并在周围加上 padding，改动不能过大，目的是使某个方向上的攻击失效

    Passive Defense 方法如果盾牌机制被泄露了还是无能为力，因为可以把比如前面的 filter 当成第一层 layer

    Proactive Defense¶

    思想是自己 attack 训练好的 model，然后把 attack 的图片加进 training data 重新训练

    反复很多次，不断的补漏洞

    但是如果 attacker 知道了我们是使用算法 A 来进行模拟，那么 attacker 可以使用算法 B来进行 attack，那么我们的 network 并不能抵御这种 attack
