    Activation Function¶

    Sigmoid Function $$ y(b+wx) = c\ \ sigmoid(b+wx)=c\frac{1}{1+e^{-(b+wx)}} $$ 对不同的 Hard Sigmoid 叠加然后加上 bias 可以实现非线性系统：

    对于一个特征来说 \(y=b+\sum_i c_i\ sigmoid(b_i+w_ix)\)

    对于多个特征来说 \(y=b+\sum_i c_i\ sigmoid(b_i+\sum_j w_{ij}x_{j})\)

    ReLU

    Rectified Linear Unit $$ y(x) = c\ max(0, x) $$ 两个 ReLU 的叠加可以得到 Hard Sigmoid

    General Guidance¶

    loss on training data：

    • large：从简单的 model 开始做来知道是 model bias 还是 optimizer 的问题
    • model bias：增加 feature，加深深度
    • optimizer：
    • small：看 loss on testing data：
    • large：
      • overfitting：增加训练资料，data augmentation，less parameters，sharingparames，less features，early stopping，regularization，dropout
      • mismatch：训练集和测试集的分布不同
    • small：好

    loss 波动很大：

    • lr 大了
    • 各个 batch 不是 homogeneous 的（即内容差别太大），不能代表整体数据，加大 batch size

    Ablation study¶

    什么是消融实验（Ablation experiment）Sandwichsauce的博客-CSDN博客

    有一点像控制变量的感觉。

    因为作者提出了一种方案，同时改变了多个条件/参数，他在接下去的消融实验中，会一一控制一个条件/参数不变，来看看结果，到底是哪个条件/参数对结果的影响更大。

    下面这段话摘自知乎，@人民艺术家：

    你朋友说你今天的样子很帅，你想知道发型、上衣和裤子分别起了多大的作用，于是你换了几个发型，你朋友说还是挺帅的，你又换了件上衣，你朋友说不帅了，看来这件衣服还挺重要的。

    Batch Normalization¶

    \[ x_{i+1}=\gamma \frac{x_i-\mu_B}{\sigma_B}+\beta \]

    其中 \(B\) 表示输入，\(\mu_B\) 表示均值，\(\sigma_B^2\) 表示方差
