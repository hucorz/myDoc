    Explainable Machine Learning¶

    简单来说就是我们需要知道 Machine 学到了什么，将 Machine 学到的东西可视化出来

    Why？进行模型诊断，有的模型准确率很高但是其实什么都没学到；模型诊断后就可以根据结果来调整

    Local Explanation¶

    Basic Idea¶

    对于输入的 x，将其分成 n 个 components \(\{x_1,...,x_n,...x_N\}\)，每个 component 是一个像素或者一个小块

    然后通过 remove 或者 modify 其中一个 component 的值，根据结果的变化判断这块对 decision 的重要性

    蓝色表示去掉那块后识别不出 True Label 了，即对结果的影响很大

    Saliency Map¶

    或者对于每个 pixel \(\{x_1,...,x_n,..,x_N\}\) 都加上 \(\Delta x\)，如果对结果影响很大，就代表很重要；其实可以用微分 \(|\frac{\partial y_k}{\partial x_n}|\)​​​ 来表示重要性，下图中越亮表示越重要

    Using a model to explain another¶

    使用一个 interpretable model 来模仿另外一个 uninterpretable model

    下图中 黑色 Black Box 是 uninterpretable model，比如 Neural Network；蓝色是 Linear Model，是 interpretable model，使他们有相似的输出

    实际上并不能使用 linear model 来模拟整个 neural network，但可以用来模拟其中一个 local region

    Local Interpretable Model-Agnostic Explanations (LIME)¶

    General¶

    1. 首先给出要 explain 的 point，代入 black box 里面

    2. 在第三个蓝色 point（我们想要模拟的区域）周围 sample 附近的 point，nearby 的区域不同，结果也会不同

    3. 使用 linear model 来模拟 neural network 在这个区域的行为

    4. 得知了该区域的 linear model 之后，我们就可以知道在该区域 x 和 y 的关系，即 x 越大，y 越小，也就 interpret 了原来的 neural network 在这部分区域的行为

    显然 nearby 的大小对结果影响很大

    LIME-Image¶

    1. 把 frog image 分成多个 segment，然后随机去掉一些 segment，把新图片丢进 black box 计算出新图片是 frog 的可能性

    2. 每个 segment 都是一个 01变量，没去掉就是 1，去掉就是 0，然后用一个 linear model 去 fit black box 的结果，得到： $$ y=w_1x_1+..+w_mx_m+...+w_Mx_M $$ \(w_i\) 表示第 i 个 segment 的权重，正值表示有正面影响，负值表示有负面影响

    Decision Tree¶

    如果我们用不限制深度的 decision tree，那么我们就可以使用 decision tree 来模拟 black box（neural network），使两者的输出相近
