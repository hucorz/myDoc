    Cuda 和 Cudnn¶

    写于 2022.4.9

    ubuntu 版本: 18.04 Desktop

    部分图片来自网络

    准备¶

    替换显卡驱动，gnome桌面在软件和更新中可以找到

    用下面的命令查看是否安装成功

    nvidia-smi

    cuda 安装¶

    cuda 下载地址：https://developer.nvidia.com/cuda-downloads，下载 runfile(local) 的版本，听说 bug 少一些

    安装 cuda

    sudo sh cuda_11.2.0_460.27.04_linux.run

    继续

    选择 accept

    这里 Driver 不要选择，途中 x 是表示选择的，其他的都选择

    安装成功

    修改环境变量

    sudo vim /etc/environment

    在末尾加入 ":/usr/local/cuda:/usr/local/cuda/bin"

    PATH="/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/games:/usr/local/games:/snap/bin:/usr/local/cuda:/usr/local/cuda/bin"

    使环境变量生效

    source /etc/environment

    查看 cuda 版本

    nvcc -V

    cuda多版本切换¶

    安装第二个 cuda 时会问你是否要建立软链接（symlink，symbolic link，符号链接），都可以，如果选择 y，cuda 就会指向新的 cuda 版本

    查看软链接

    stat cuda   # /usr/local 目录下

    要切换 cuda 版本只需要切换 cuda 文件的软链接就行

    sudo rm -rf cuda
sudo ln -s /usr/local/cuda-9.1 /usr/local/cuda

    cudnn 下载¶

    官网登陆后即可下载， https://developer.nvidia.com/rdp/form/cudnn-download-survey 

    # 解压
tar -xvf cudnn-10.2-linux-x64-v7.6.5.32.tgz
# 复制
sudo cp cuda/include/cudnn.h    /usr/local/cuda/include/
sudo cp cuda/lib64/libcudnn*    /usr/local/cuda/lib64/
# 验证是否成功
cat /usr/local/cuda/include/cudnn.h | grep CUDNN_MAJOR -A 2
