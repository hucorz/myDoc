　　int x;     
　　bool operator < (const node& a) const     {

　　}

//

    algorithm¶

    \(include<algorithm>\)

    lower_bound()¶

    二分查找

    对于升序数组：

    lower_bound( begin, end, num)：查找第一个大于等于 num 的数并返回其地址，不存在返回end。返回值减去begine即索引下标

    upper_bound( begin, end, num)： 大于

    对于降序数组：需要重载

    //升序 a数组中 k 的个数
upper_bound(a, a+n, k)-lower_bound(a, a+n, k);
